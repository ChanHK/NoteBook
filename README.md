# NoteBook
This is a personal project which allows the user to store notes. 

## Develop
This project uses MEAN stack:
- mongodb
- expressjs
- angular
- nodejs

## Compile
Front-end
```npm
ng serve
```

Back-end
```npm
nodemon server
```

Full stack
```npm
npm run dev
```
